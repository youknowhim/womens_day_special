🌸 Women’s Day Special Web Application

A modern React-based web application built to celebrate International Women’s Day by highlighting women's achievements, promoting gender equality awareness, and using AI-powered tools to educate users.

This platform combines informational content and AI-driven features to inspire users and encourage conversations about women’s empowerment.

🚀 Features
🌟 Women Achievers Explorer

Discover inspiring women from different regions.
Users can enter a country and state, and the application generates top women achievers from that region using AI.

🧠 Gender Bias Detector

An AI-powered tool that analyzes sentences to detect gender bias or stereotypes and suggests more inclusive alternatives.

Example:

Input:
Women are bad leaders

Output:
This sentence contains gender bias. Leadership ability is not determined by gender.
✨ Women Facts Generator

Generates 5 interesting facts about women’s achievements in fields such as science, leadership, sports, and history using AI.

Example facts:

Marie Curie was the first person to win two Nobel Prizes.

Kalpana Chawla was the first woman of Indian origin in space.

Mary Kom is a six-time world boxing champion.

📚 Women Empowerment Blog

A collection of in-depth articles about women’s empowerment, covering topics such as:

Gender equality

Women in leadership

Women in science

Women in sports

Education for girls

Each blog is presented in large article cards with images for better readability.

🏠 Engaging Home Page

The homepage provides a welcoming introduction with:

Women empowerment message

Feature navigation cards

Inspirational images
